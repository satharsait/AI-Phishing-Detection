# AI Phishing Detection System

An AI-based phishing detection system developed as a BCA academic project using cybersecurity and machine learning concepts.

## Features

* Detects phishing URLs using machine learning
* User authentication system
* Dashboard and history tracking
* Flask-based web application
* Interactive frontend with HTML, CSS, and JavaScript

## Technologies Used

* Python
* Flask
* Machine Learning
* HTML
* CSS
* JavaScript
* SQLite

## Project Structure

* `app.py` – Main Flask application
* `model.pkl` – Trained machine learning model
* `templates/` – HTML frontend pages
* `static/` – CSS and JavaScript files
* `instance/` – Database and application data

## Purpose

This project was developed to explore how machine learning can be integrated with cybersecurity to identify phishing attempts and improve online security awareness.

## Future Improvements

* Real-time URL scanning
* API integration for threat intelligence
* Improved machine learning accuracy
* Browser extension support
* Cloud deployment
